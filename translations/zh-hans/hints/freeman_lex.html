用这两种情况写一个single line 条件语句if 。 

return ... if n> 1 else ...
下一步该怎么做？

对于n 0 或 1，您的函数必须返回 1。 

return ... if n> 1 else 1
我还需要提示。

差不多就是这样。如果当前n 仍大于 1，则返回n 的乘积，并调用带有n-1 参数的同一函数（递归方法L 多次嵌套调用同一函数）。 

return n*factorial(n-1) if n> 1 else 1
那么，我的最终解决方案应该是怎样的呢？

你只需将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

以下是整个解决方案：

def factorial(n: int) -> int:

    return n*factorial(n-1) if n> 1 else 1

从 1 开始，乘以从 2 到给定n 的所有整数。 

def factorial(n: int) -> int:

    res = 1
    for i in range(2, n + 1):
        res *= i
        
    return res
